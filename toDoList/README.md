# toDoList (learning project)

A simple to-do list console application used to practice C# basics and simple state management.

What it is
- A small command-line to-do app for adding, listing, and removing tasks. Great for practicing program flow and basic persistence patterns.

Prerequisites
- .NET 9 SDK. Check with `dotnet --version`.

Build & run
From the repository root:

```bash
dotnet build toDoList
dotnet run --project toDoList
```

Notes
- This is a learning/example project. You can extend it with unit tests, file-based persistence, or a simple menu-driven UI.
